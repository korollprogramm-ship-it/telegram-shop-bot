# 🤖 Telegram Shop Bot — Готовый интернет-магазин

Полноценный Telegram-бот с Mini App интернет-магазином. Всё работает "из коробки"!

## ✨ Функции

- 🛍️ **Каталог товаров** с категориями
- 🛒 **Корзина** и оформление заказа
- 👑 **Админ-панель** для управления товарами
- 📦 **Просмотр заказов** в реальном времени
- 🔗 **Mini App** — магазин прямо в Telegram
